Sales Dashboard with Power BI

Barbas Bellfires had multiple data sources for all their sales-related information. The goal of this project was to create a single source of thruth for all sales-related information, meanwhile being accessible and easy to use for all dealermanagers. Given the organization's standardization on the Office 365 platform, Power Bi was the right tool for this project. With the dealermanagers we've defined the need for four dashboards.

  1. Sales Intake: A dashboard that visualizes the ordered fireplaces over time.
  2. Budget Realisation: A dashboard that visualizes the realised sales versus the defined targets.
  3. Showroom Appliances: A dashboard that visualizes all dealers with their installed appliances in their showrooms
  4. Sales Comparison: A dashboard that empowers the dealermanagers to compare sales data based on two variable timeframes

Note that all datasets have been modified for demo purposes, but still, take a look at and interact with (a representation of) the end result I've created for the dealermanagers of Barbas Bellfires.
